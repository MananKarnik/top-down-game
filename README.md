# top-down-game

Prototype for a Top Down Game using Pygame

## Clone repository

```sh
git clone https://github.com/MananKarnik/top-down-game.git
```

## Install dependencies

```sh
pip install -r requirements.txt
```

## Run the game

### Change into project directory

```sh
cd top-down-game
```

### Execute main.py

```sh
python src/main.py
```
